Action Guide

Salesperson Actions

>

New lead: Open CRM Leads > Click Add Lead > Complete the popup > Set New Contact > Set urgency/source > Set next follow-up.

>

Lead replies: Search the lead > Update the table directly > Change status > Set the next follow-up.

>

Parent replies: Find the lead in the table > Click Edit > Update the popup profile and notes > Set next follow-up.

>

Parent asks time: Open Weekly Timetable/Teacher Overview > Filter slots > Propose open slots only > Record options in notes.

>

Assessment: Set Assessment > Weekly Timetable opens > Choose teacher slot > Confirm assessment.

>

Trial: Set Trial Class > Weekly Timetable opens > Choose teacher slot > Confirm trial > Follow up after both trial dates pass.

>

No reply: Use no-reply template > Set Follow Up or No Response > Set next follow-up date. If clearly closed, set Lost or Not Interested.

>

Enrollment: Set Enrolled > Open Students > Mark Registered > Add package and regular class slots.

Admin Actions

>

Teacher setup: Open Teachers > Add profile > Add subjects/tier/rate > Add timetable slots > Save.

>

Student regular slot: Open Students > Edit student > Add teacher/day/time/subject > Save.

>

Student not show: Click class > Set Student Not Show > Add remark > Save. It turns red and pays RM5 allowance.

>

Class cancelled: Click class > Choose Cancelled/Public Holiday/Teacher Leave > Add remark > Save. It turns grey and is not counted for payout.

>

Undo wrong cancellation: Open Replacement Classes > Find the case > Click Undo Cancellation > Original class returns to booked.

>

Teacher partial leave: Mark only affected class slots as Teacher Leave.

>

Teacher payout: Open Reports > Select teacher/month/year > Generate report > Export PDF.

>

Before important edits: Pull From Sheet > Edit > Save > Confirm auto-sync or Push manually.
