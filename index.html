Sales Guide

  1. Add every new parent contact in CRM Leads, including parent phone, child name, source, urgency, and interested subjects.
  2. Set status as New Contact first, then update it after every WhatsApp/call.
  3. Always write a short note after speaking with the parent and set the next follow-up date before leaving the lead.
  4. Use Weekly Timetable or Teacher Overview to check available teacher slots before proposing times.
  5. When parent agrees to assessment, choose child, date, time, teacher, and subject, then click Book Assessment.
  6. When parent agrees to trial class, choose child, date, time, teacher, and subject, then click Book Trial.
  7. Assessment and trial bookings block the teacher's 30-minute slot and show in the timetable as 25-minute sessions.
  8. After the child enrolls, update lead status to Enrolled and create/update the Student Profile with package and regular class slots.
